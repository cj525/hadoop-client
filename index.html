hadoop-client

The Tribe needed a client wrapper for its shiny new HBase Stargate installation, and since a good one didn't exist, at least not for .NET, we wrote one! Since then, our efforts have expanded into the broader world of Hadoop, and we're hoping to dig up some really useful insights along the way. This site is all about those efforts.

Content

  • Check out the hbase-client blog for the latest news and insights into hbase-client.
  • Community content is available on the hbase-client wiki.
  • Interactive Visio Diagrams will be made available as they are created.
  • API Documentation is generated using Doxygen.
  • Unit Test Coverage is analyzed using dotCover.
  • Feature-level Unit Test Documentation is generated using Pickles.
  • Dependencies and Complexity is analyzed using NDepend.
